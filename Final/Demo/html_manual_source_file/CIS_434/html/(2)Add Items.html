Adding Items to the Menu

Here is window that allows users to add items to the menu. There are four required inputs: Name, Price, and Quantity, and Category. The image is optional. There are four available categories: Appetizers, Entree, Drinks, and Dessert.

Add Item Steps

    1. Enter the item name

    2. Enter the item price

    3. Enter the item quantity

    4. Select the item category

    5. OPTIONAL: Select upload to add an image for the item

    6. Click Save

Once the item is added succesfully, there will be a confirmation message and the input fields will clear. If the item was not added succesfully a message will appear to fix any errors.
